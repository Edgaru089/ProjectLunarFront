# ProjectLunarFront

A simple online judge system indended to be used under a local and safe network, which is a rather ambitionous project.

简易的在线测评系统，目标是在安全的局域网中使用。好大的一个坑啊。

LunarPhase框架被大量修改以适应SFML，添加了高效的线程池，但还是那么低效。
